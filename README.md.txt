# Fin-Lumen Tracker

A personal investment decision dashboard combining:

- Fundamentals (FS)
- Cycle Alignment (CA)
- Astro Timing (AT)
- Risk (DZ)

## Features

- Track up to 100 stocks
- Manual update per stock
- Weekly "Update All"
- Decision engine (Action + Positioning)
- Astro timing layer (in progress)

## Tech Stack

- Next.js (frontend + backend)
- Vercel (deployment)
- Future:
  - Supabase (database)
  - Swiss Ephemeris (astro engine)
  - Market data API

## Usage

1. Click "Update All" weekly
2. Use individual update for decision checks
3. Monitor Phase / Action / AT
